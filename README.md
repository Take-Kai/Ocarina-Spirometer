# オカリナ型肺活量測定器

健康維持を促すためのオカリナ型をした児童向け肺活量測定器．

八戸高専のカリキュラムである「創成実験」の中で本科4年次にチームで制作．

私はオカリナの運指と息を吹くことで音を鳴らす機構とコード，息量の測定プログラム作成を担当．

「吹く」という動作で共通している楽器と肺活量測定を組み合わせ，健康維持と楽しさを両立するシステムがあれば面白いという考えに基づいて制作．

詳細は[Notionプロジェクトページ](https://wholesale-beginner-8e9.notion.site/347306a87b4980d481f2d9815bac4a0a?v=347306a87b498023832d000cf5d8d356&p=347306a87b4980368caac878c461c280&pm=c)を参照してください．

## ⚒️ 開発環境
- **MCU**：Arduino ATmega328P
- **Sensor**：BME680気圧センサ
- **Hardware**：タクトスイッチ×8, 圧電スピーカ

## 💻 アルゴリズム
本システムでは，「オカリナの運指と呼気によって音を出す」と「呼気の強弱によって肺活量を3つの指標で評価する」を組み合わせている．

### 1. 呼気検知
起動時の気圧を5回サンプリングして平均値(`press_act_ave`)を算出．

サンプリングした気圧値（基準）からの気圧上昇を検知して「吹いた」と判定．
- **演奏できる気圧**：基準気圧 + 0.04 hPa 以上
- **肺活量が「良い」となる気圧**：基準気圧 + 0.5 hPa 以上 + 湿度上昇

### 2. 運指の判定
8つのデジタル入力(`buttonPin_1` 〜 `8`)の状態を組み合わせ，オカリナ特有の運指を再現．

| 判定音 | ボタンの状態（組み合わせ例） | 周波数 [Hz] |
| :--- | :--- | :--- |
| 低いド | 全てのボタンがLOW（全て押さえる） | 262hz |
| ソ | 左手4番HIGH, 右手4番LOW | 392Hz |
| 高いド | 右手小指以外がHIGH | 523Hz |
